# Flux-Shop

## Project Overview

Flux-Shop is a full-featured e-commerce application built with modern technologies.

## Languages and Technologies Used

- **Languages**: JavaScript, Python, TypeScript, Rust
- **Frontend**:
  - React
  - Vite
  - ESLint
  - Other libraries:
    - axios
    - react-router-dom
    - react-leaflet/leaflet
    - swiper
    - react-toastify
    - react-icons
    - animate.css
    - @react-oauth/google
    - @marsidev/react-turnstile

- **Backend**:
  - Django
  - Django REST Framework
  - SimpleJWT
  - CORS headers

- **Database**: PostgreSQL

- **Hosting**:
  - Gunicorn
  - Nginx

- **Containerization**:
  - Docker
  - docker-compose

- **CI/CD**:
  - GitHub Actions workflows for Android and Desktop builds

## Getting Started

### Running Locally with Docker
1. Clone the repository:
   ```bash
   git clone https://github.com/Xserio617/Flux-Shop.git
   cd Flux-Shop
   ```
2. Build the Docker containers:
   ```bash
   docker-compose build
   ```
3. Start the application:
   ```bash
   docker-compose up
   ```

### Running Locally without Docker
1. Install the necessary dependencies:
   ```bash
   npm install
   ```
2. Run the application:
   ```bash
   npm start
   ```
